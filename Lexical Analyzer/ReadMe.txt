编程语言为Java，使用JDK1.7，IDE为Eclipse Luna，编码为UTF-8

程序基本执行过程：
1、读取RE定义文件，对正规表达式进行预处理（加点，中缀变后缀）
2、RE -> NFA
3、NFA -> DFA
4、DFA -> 词法分析表
4、读取程序文件，对程序文件进行预处理（处理空白字符）
5、根据词法分析表分析程序，生成TOKEN序列和符号表并保存

程序说明：
1、目录中的Regular.re是正轨表达式定义文件，其中每一行是一个正规表达式，定义规则为：
返回TOKEN类型______对应正规表达式
注意：如果正规表达式中需要使用有实在意义的. ( ) | * 空格 字符，需要用\. \( \) \| \空格 进行转义

2、目录中的SourceCode是需要分析的程序源代码文件，生成的Token.txt，Symbol.txt分别是分析得到的Token序列和符号表。

3、目前定义的Re是简化的Java语言，可以识别Java保留字（例如public），标识符（例如apple），只含有大小写字母和数字的字符串（例如"aadsf344GGG"），大小写字母或数字的字符（例如'a'），符号（例如,），整数（例如123），小数（例如3.14），布尔型（例如true）；亦可以根据实际情况自己定义正规表达式
